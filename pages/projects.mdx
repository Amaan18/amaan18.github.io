# Projects

### TsAlgo

[GitHub](https://github.com/amaan18/tsalgo) | [NPM](https://npmjs.com/tsalgo)

- Designed an open-source library of typed implementations of data structures, utility functions, and common algorithms
  for JavaScript/TypeScript. Wrote in TypeScript using generics, and with 0 dependencies

### Image Repository API

[GitHub](https://github.com/amaan18/image-repository-api)

- Built an Image Repository API with Typescript (NestJS), MongoDB, and AWS S3 buckets for secure image storage
- Included authentication using JWTs, role-based authorization and guards, and easy setup using Docker containers
- Adhered to SOLID design principles and used NestJS's dependency injection architecture to achieve inversion of control

### Github Battle

[GitHub](https://github.com/amaan18/github-battle) | [Demo](https://react-git-battle.netlify.app)

- Built a Web App that displays the most popular repositories on Github and allows the users to 'Battle' github profiles
- Used React, Redux, and the GitHub API to fetch user data

### Search Algorithm Visualizer

[GitHub](https://github.com/amaan18/a-star-pathfinding) | [Demo](https://astar-visualization.netlify.app)

- A visualizer for the A\* Search Algorithm implemented using a Binary Heap to optimize access times
- Used p5.js for visualization

### Campground Review Web app

[GitHub](https://github.com/amaan18/yelpcamp) | [Demo](https://whoevenlikescamping.herokuapp.com)

- Full stack Web App for community based Campground Reviews. CRUD functionality with User Authentication, and use of the Google Maps API
- Used Node.js, Express, MongoDB, and EJS templating
